# Jason Horsley Tech Homepage

Homepage for Jason Horsley, mostly a resume, maybe a repo for future projects.

## Setup

* sudo apt-get install tasksel
* sudo tasksel install lamp-server
* cd /var/www/
* sudo rm -r html
* git clone https://github.com/JasonHorsleyTech/JasonHorsleyTech-Home .

## Development

#### CSS made with SCSS, compile with

* cd /var/www/html
* sass --watch scss:css

#### Everything else vanilla
